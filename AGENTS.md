# Agent Guidance

This repository uses Architecture Decision Records (ADRs) for approved engineering decisions.

Accepted engineering decisions live under `docs/adr/`. AI coding agents and human contributors should treat accepted ADRs as authoritative repository guidance.

Draft engineering discussions live under `docs/adr/drafts/`. Draft ADRs are discussion material only and are not repository policy.

New repository-wide standards should begin as draft ADRs. Once a decision is explicitly approved, promote the final decision into the next numbered ADR under `docs/adr/`.

Do not duplicate ADR contents in this file. Link to or reference accepted ADRs when repository guidance depends on a specific decision.
